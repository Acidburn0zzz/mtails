Spenden

Diese Version behebt zahlreiche Sicherheitslücken. Alle Benutzer sollten so schnell wie möglich aktualisieren.

  1. Änderungen
  2. Bekannte Probleme
  3. Holen Sie sich Tails 2.7
  4. Was kommt als Nächstes?

Änderungen

Aktualisierungen und Änderungen

  • Aktualisierung von Tor auf 0.2.8.9.

  • Aktualisierung des Tor Browser auf 6.0.6.

  • Aktualisierung von Icedove auf 45.4.0.

Behobene Probleme

  • Die Unterstützung für mehrfache Architekturen in Synaptic wurde behoben. Dies sollte defekte Installation von zusätzlichen Softwarepaketen über Synaptic beheben. (#11820)

  • Die voreingestellte Rechtschreibprüfung von Icedove wurde von Französisch auf Englisch umgestellt. (#11037)

Lesen Sie für mehr Details unser Änderungsprotokoll.

Bekannte Probleme

  • Benutzer, die das Sicherheitslevel des Tor Browser auf Hoch stellen, müssen auf einen Link klicken, um das Ergebnis einer Suche, die mit dem Suchfeld durchgeführt wurde, zu sehen.

Lesen Sie sich die Liste der längerfristig bekannten Probleme durch.

Holen Sie sich Tails 2.7

  • Befolgen Sie zum Installieren unsere Installationsanweisungen.

  • Es ist eine automatische Aktualisierung von 2.6 auf 2.7 verfügbar.

    Falls Sie keine automatische Aktualisierung durchführen können oder das Starten nach einer automatischen Aktualisierung fehlschlägt, versuchen Sie bitte eine manuelle Aktualisierung.

  • Tails 2.7 herunterladen

Was kommt als Nächstes?

Tails 2.8 ist für den 13. Dezember geplant.

Werfen Sie einen Blick auf die Roadmap, um zu sehen, was wir als Nächstes vorhaben.

Wir brauchen Ihre Hilfe und es gibt viele Wege, Tails zu unterstützen, (Spenden ist nur einer davon). Sprechen Sie uns an!
