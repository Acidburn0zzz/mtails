Mudanças

Atualizações e mudanças

  • Tor Browser foi atualizado para a versão 7.5.3.

  • Thunderbird foi atualizado para 52.7.0.

  • Intel microcode foi atualizado para 3.20180312.1, que adiciona mitigação para o Spectre variante 2 para os processadores Intel mais comuns.

  • Linux foi atualizado para 4.15.11-1.

Problemas resolvidos

  • Os caracteres de entrada em japonês e coreano voltaram a funcionar no Navegador Tor.

  • O Navegador Tor foi corrigido para que extensões instaladas por usuários, tais como Tails Verification, funcionem novamente.

Para mais detalhes, leia nosso changelog.

Veja a lista de problemas de longa data.

Baixe o Tails 3.6.2

  • Para instalar, siga nossas instruções de instalação.

  • Atualizações automáticas do Tails 3.6 e 3.6.1 para o 3.6.2 estão disponíveis.

    Se você não conseguir fazer uma atualização automática ou se seu Tails não iniciar após uma atualização automática, tente uma instalação manual

    Se você estiver fazendo uma atualização manual a partir do Tails 3.2, 3.3, 3.4 ou 3.5, então somente será possível selecionar uma imagem ISO se você estiver rodando o Tails em inglês. Nos outros idiomas, o botão de escolha de arquivo ficará inativo:

    Para fazer uma atualização manual do Tails 3.2, 3.3, 3.4 ou 3.5, você pode:

    • Reinicie o Tails em inglês.

    • Inicie o Instalador Tails em inglês a partir da linha de comando:

      LANG=en_US tails-installer

  • Baixe o Tails 3.6.2.
