Esta versión corrige muchos problemas de seguridad y los usuarios deberían actualizarse lo antes posible.

  1. Cambios
  2. Problemas conocidos
  3. Obtener Tails 3.1
  4. ¿Qué novedades hay?

Cambios

Actualizaciones y cambios

  • Actualizar Tor Browser a 7.0.4.
  • Actualizar Linux a 4.9.30-2+deb9u3.

Problemas arreglados

  • Asegúrate de que Thunderbird borra su directorio temporal, que contiene por ejemplo archivos adjuntos abiertos en el pasado. #13340

  • Corregir traducciones de sincronización horaria y de las notificaciones "Tor is ready". #13437

Para más detalles, lee nuestro changelog.

Problemas conocidos

Nada concreto para esta versión.

Mira la lista de problemas duraderos.

Obtener Tails 3.1

  • Para instalarlo, sigue nuestras instrucciones de instalación.

  • Para actualizar, las actualizaciones automáticas están disponibles de 3.0 a 3.1. Debido al bug #13426, las actualizaciones automáticas de 3.0.1 están desactivadas. Si no puedes realizar una actualización automática o si no se inicia tras una actualización automática, intenta hacer una [[actualización manual.

  • Bajar Tails 3.1.

¿Qué novedades hay?

Tails 3.2 está planeado para el 3 de octubre.

Echa un vistazo a nuestro roadmap para ver hacia dónde nos dirigimos.

Necesitamos tu ayuda y hay muchas maneras decontribuir a Tails (donar sólo es una de ellas). Ven habla con nosotros!
