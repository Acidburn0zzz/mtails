This release fixes many security issues and users should upgrade as soon as possible.

  1. Mudanças
  2. Problemas conhecidos
  3. Get Tails 2.7
  4. O que vem por aí?

Mudanças

Atualizações e mudanças

  • Upgrade Tor to 0.2.8.9.

  • Upgrade Tor Browser to 6.0.6.

  • Upgrade Icedove to 45.4.0.

Problemas resolvidos

  • Fix multi-architecture support in Synaptic. This should fix broken installations of additional packages from Synaptic. (#11820)

  • Set default spell checking language of Icedove to English instead of French. (#11037)

Para mais detalhes, leia nosso changelog.

Problemas conhecidos

  • Users setting their Tor Browser security slider to High will have to click on a link to see the result of the search they done with the search box.

Veja a lista de problemas de longa data.

Get Tails 2.7

  • Para instalar, siga nossas instruções de instalação.

  • To upgrade, an automatic upgrade is available from 2.6 to 2.7.

    Se você não conseguir fazer uma atualização automática ou se seu Tails não iniciar após uma atualização automática, tente uma instalação manual

  • Download Tails 2.7.

O que vem por aí?

Tails 2.8 is scheduled for December 13.

Confira o nosso roadmap e veja nossos objetivos futuros.

We need your help and there are many ways to contribute to Tails (donating is only one of them). Come talk to us!
