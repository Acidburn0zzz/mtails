Donate

你有很多種向Tails做貢獻的途徑。 任何嘗試都不會微不足道，且無論你給社區帶來什麼都會被認可。 所以請繼續讀下去，看看有沒有你給Tails帶來不一樣的地方。

分享你的經驗

每個用戶都能幫助他人，或向開發者提供有用的信息。

  • Report bugs
  • Test experimental ISO images
  • Provide input to developers
  • Help other Tails users

捐款

Donating speeds up the development of Tails.

  • Make a donation

貢獻你的語言技能

寫手

好的寫手能讓 Tails 得到更多的人認識。

  • 改進文檔
  • 寫新聞通稿

翻譯人員

譯者能讓世界上更多的人使用 Tails。

  • 用你的母語裝備 Tails

演講者

做演講可以向所有不同類型的公衆宣傳 Tails。

  • 在活動中宣講

貢獻你的計算機技能

開發者或維護者

具有多種不同技能的軟件專業人士可以改進 Tails。

  • 在源代碼上工作
  • 通過在 Debian 上工作來改進 Tails

系統管理員

系統管理員可以爲 Tails背後的基礎設施做貢獻。

  • 運行一個 HTTP 鏡像
  • 改進 Tails 的基礎設施

設計師

Designers can make Tails easier to use and more appealing.

  • Improve the website
  • Design graphics
  • Improve the Tails user experience
