Search the documentation

Read the official documentation to learn more about how Tails works and maybe start answering your questions. It contains:

  • General information about what Tails is
  • Information to understand how it can help you and what its limitations are
  • Guides covering typical uses of Tails

Visit Tails documentation

Learn how to use Tails

The Tails project itself lacks time and skills to provide training on how to use Tails but other organizations do so.

Frequently asked questions

Search our list of frequently asked questions.

Upgrade

Make sure you are using the latest version, as upgrading might solve your problem.

Check if the problem is already known

You can have a look at:

  • The list of known issues
  • The list of things that will be in the next release
  • The rest of our open tickets on Redmine

Report an error

If you are facing an error in Tails, please follow the bug reporting guidelines.

If Tails does not start, please see our specific reporting guidelines.

Request a feature

If you would like to see a new feature in Tails, search the open tickets in Redmine first, and file a new ticket in there if no existing one matches your needs.

Get in touch with us

Email

Mail us on our private mailing list:

tails-support-private@boum.org

We answer requests in:

  • English
  • French
  • Spanish
  • Italian

Requests not in English might take longer to answer. Imperfect English is welcome :)

Encrypting your emails with our OpenPGP key is the only way to achieve end-to-end encryption.

Chat

Join our XMPP chat room to talk to contributors and users.

  • server: conference.riseup.net
  • room: tails
  • use TLS/SSL to connect!
