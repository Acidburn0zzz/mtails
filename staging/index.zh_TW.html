Tails is a live operating system that you can start on almost any computer from a USB stick or a DVD.

它旨在保留你的隱私 以及 網路匿名 並且可以助你：

  • 匿名地瀏覽網際網路 和規避網路審查;
    所有欲連到網際網路的連線都會透過 Tor Netzwerk 系統；
  • 在使用的電腦上 不會留下㾗跡記錄, 除非用戶刻意這麼要求
  • 使用最先進的密碼學工具 來加密檔案、電子郵件以及即時訊息。
