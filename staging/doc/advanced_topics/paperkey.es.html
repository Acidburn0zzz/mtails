Paperkey es una herramienta de línea de comandos para exportar claves secretas OpenPGP en un formato adecuado para imprimir en papel.

El papel se puede destruir fácilmente, pero también tiene unas cualidades de retención increíblemente largas, mucho más largas que los soportes magnéticos u ópticos que se utilizan generalmente para realizar copias de seguridad de datos informáticos. Así que paperkey puede ser útil en combinación con otras estrategias de copia de seguridad.

Tu clave OpenPGP tal y como fue exportada por paperkey aún está protegida por tu contraseña.

Para aprender a usar paperkey, lee la documentación en la web de paperkey.

Por ejemplo, para exportar una clave secreta OpenPGP usando paperkey, ejecuta el siguiente comando, reemplazando [keyid] con el ID de la clave que deseas exportar:

gpg --export-secret-key [keyid] | paperkey | gedit
