GNOME Boxes intenta proveer una interfaz simple para crear y utilizar maquinas virtuales para Linux con GNOME. GNOME Boxes no permite utilizar un volumen persistente.

Ejecutar Tails dentro de una máquina virtual tiene bastantes implicaciones de seguridad. Dependiendo en el sistema operativo base y tus necesidades de securidad, ejecutar Tails en una máquina virtual podría ser peligroso.

Las instrucciones siguientes han sido probadas en Debian Jessie.

Instalación

Para instalar GNOME Boxes en Debian o Ubuntu ejecuta la siguiente orden:

sudo apt install gnome-boxes

Ejecutar Tails desde una imagen ISO

  1. Inicia GNOME Boxes.
  2. Da click en el botón Nuevo en la parte superior de la ventana.
  3. In the Source Selection dialog, choose Select a file and browse for the ISO image that you want to start from.
  4. In the Review dialog, click on the Create button on the top of the window.

Portapapeles compartido

The shared clipboard of GNOME Boxes is enabled by default. This can allow sensitive data to be copied by mistake from the virtual machine onto the host operating system or vice versa.

We recommend you to disable the shared clipboard.

Para inhabilitar el portapapeles compartido:

  1. Choose  ▸ Properties.

  2. Deselect the Share Clipboard option.
