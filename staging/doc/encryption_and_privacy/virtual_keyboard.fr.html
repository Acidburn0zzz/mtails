Faire un don

Si vous pensez que l'ordinateur que vous utilisez n'est pas digne de confiance, par exemple s'il s'agit d'un ordinateur public dans une bibliothèque, tout ce que vous tapez pourrait être enregistré par un enregistreur de frappe matériel.

Vous pouvez utiliser le clavier visuel de GNOME pour vous protéger des enregistreurs de frappe matériels lorsque vous tapez des mots de passe et du texte sensible. Pour afficher le clavier visuel, consultez la documentation correspondante.
