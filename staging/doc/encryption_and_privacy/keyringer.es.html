Keyringer es un programa para compartir secretos encriptados distribuidos que funciona desde la línea de comandos.

Keyringer te permite gestionar y compartir secretos usando OpenPGP y Git con comandos personalizados para cifrar, descifrar, y editar archivos de texto y otros tipos de documentos. Guardando estos secretos en Git, puedes compartirlos con otras personas en una manera distribuida.

Usar keyringer requiere conocimiento previo de Git y la línea de comandos.

Para aprender a usar keyringer, lee la documentación en el sitio web de keyringer.

Puedes usar el comando open de keyringer para editar, encriptar y compartir documentos LibreOffice, imágenes, etc.

To store your keyringer configuration across separate working sessions, you can enable the Dotfiles persistence feature and make persistent the files in the .keyringer folder of your Home folder.

Por ejemplo, si tienes un simple keyringer llamado top-secret:

/live/persistence/TailsData_unlocked/dotfiles
└── .keyringer
    ├── config
    └── top-secret

Make sure to update your dotfiles each time you use the init, teardown, destroy, or preferences command of keyringer.

Para realizarlo puedes ejecutar el siguiente comando:

rsync -va --ignore-existing --delete ~/.keyringer /live/persistence/TailsData_unlocked/dotfiles
