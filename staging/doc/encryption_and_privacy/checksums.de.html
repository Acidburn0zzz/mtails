Spenden

GtkHash erlaubt es Ihnen Prüfsummen zu berechnen. Prüfsummen können nützlich sein, um die Integrität einer Datei zu verifizieren, beispielsweise falls Sie diese aus dem Internet heruntergeladen haben.

Um die Prüfsumme einer Datei zu berechnen:

  1. Open the file browser, either from the Places menu or the Home icon on the desktop.

  2. Navigieren Sie zu dem Ordner, der die Datei enthält, für die Sie die Prüfsumme berechnen wollen.

  3. Rechtsklicken Sie die Datei und wählen Sie Eigenschaften aus.

  4. Klicken Sie im Eigenschaften-Dialogfenster auf den Digests-Reiter.

  5. Wählen Sie in der Hash Funktion Spalte des Digests-Reiter-Bereichs, die Ankreuzfelder aus, die den Prüfsummen entsprechen, die Sie berechnen möchten.

  6. Klicken Sie auf die Hash-Schaltfläche.

  7. Die berechneten Prüfsummen erscheinen in der Digest-Spalte.
