Faire un don

GtkHash vous permet de calculer des sommes de contrôle. Les sommes de contrôle peuvent servir à vérifier l'intégrité d'un fichier, par exemple si vous le téléchargez depuis l'Internet.

Pour calculer la somme de contrôle d'un fichier :

  1. Ouvrez le navigateur de fichiers, soit depuis le menu Emplacements, soit depuis l'icône Dossier personnel sur le bureau.

  2. Allez dans le dossier contenant le fichier dont vous voulez calculer la somme de contrôle.

  3. Faites un clic-droit sur le fichier et choisissez Propriétés.

  4. Dans la boîte de dialogue Propriétés, cliquez sur l'onglet Résumés.

  5. Dans la colonne Fonction de hachage de l'onglet Résumés, sélectionnez les cases à cocher correspondant aux sommes de contrôle que vous voulez calculer.

  6. Cliquez sur le bouton Hachage.

  7. La somme de contrôle calculée apparaît dans la colonne Résumé.
