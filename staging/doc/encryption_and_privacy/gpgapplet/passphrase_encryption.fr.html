Avec l'applet OpenPGP vous pouvez chiffrer du texte avec une phrase de passe en utilisant la chiffrement par phrase de passe d'OpenPGP.

Écrire un texte confidentiel dans un navigateur web n'est pas sûr car des attaques JavaScript peuvent y accéder depuis ce même navigateur. Vous devriez plutôt écrire votre texte dans une autre application, le chiffrer en utilisant l'applet OpenPGP, et coller votre texte chiffré dans votre navigateur, avant de l'envoyer par courrier électronique par exemple.

Lorsque vous utilisez l'applet OpenPGP pour chiffrer des courriers électroniques, les caractères non-ASCII (par exemple les caractères non-Latin ou avec des accents) peuvent ne pas s'afficher correctement pour le destinataire du courrier électronique.

Si vous devez souvent chiffrer des courriers électroniques, nous vous recommandons de plutôt configurer Thunderbird.

Cette méthode nécessite que vous partagiez la phrase de passe secrète avec les personnes qui déchiffreront ce texte. OpenPGP permet également l'utilisation de la cryptographie par clé publique pour envoyer des messages confidentiels sans avoir à partager une phrase de passe. Voir la documentation correspondante.

  1. Écrivez votre texte dans un éditeur de texte. Ne l'écrivez pas dans le navigateur web !

    Cliquer sur l'Applet OpenPGP et choisir Ouvrir l'Éditeur de Texte pour ouvrir gedit.

  2. Sélectionnez avec la souris le texte que vous voulez chiffrer. Pour le copier dans le presse-papier, faire clic droit sur le texte sélectionné et choisir Copier dans le menu.

    L'applet OpenPGP affiche désormais des lignes de texte, signifiant que le presse-papier contient du texte non-chiffré :

  3. Cliquez sur l'applet OpenPGP et choisir Chiffrer le presse-papier dans le menu.

    Si vous obtenez le message d'erreur “Le presse-papier ne contient pas de données valides.”, re-essayez de copier votre texte, depuis l'étape 2.

  4. Dans la boîte de dialogue Phrase de passe, tapez la phrase de passe de votre choix. Tapez de nouveau cette phrase de passe dans la seconde boîte de dialogue.

  5. L'applet OpenPGP affiche désormais un cadenas, signifiant que le presse-papier contient du texte chiffré :

  6. Pour coller le texte chiffré dans une autre application, faire clic droit dans l'application où vous voulez le coller et choisir Coller dans le menu.

    Par exemple, vous pouvez le coller dans le navigateur web pour l'envoyer par courrier électronique.

Vous pouvez également déchiffrer un texte chiffré avec une phrase de passe en utilisant l'applet OpenPGP.
