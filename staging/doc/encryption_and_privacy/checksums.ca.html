GtkHash allows you to calculate checksums. Checksums can be useful to verify the integrity of a file, for example, if you downloaded it from the Internet.

To calculate the checksum of a file:

  1. Open the file browser, either from the Places menu or the Home icon on the desktop.

  2. Navigate to the folder containing the file that you want to calculate the checksum for.

  3. Right-click on the file and choose Properties.

  4. In the Properties dialog, click on the Digests tab.

  5. In the Hash Function column of the Digests tabbed section, select the check boxes corresponding to the checksums that you want to calculate.

  6. Click on the Hash button.

  7. The calculated checksums appear in the Digest column.
