Si crees que la computadora que estas usando no es confiable, por ejemplo si usas una computadora pública en una biblioteca, todo lo que escribas puede ser grabado por un keylogger de hardware.

Puedes usar el GNOME screen keyboard para protegerte contra un keylogger de hardware cuando escribes contraseñas y texto delicado. Para mostrar el teclado en pantalla, lee la documentación correspondiente.
