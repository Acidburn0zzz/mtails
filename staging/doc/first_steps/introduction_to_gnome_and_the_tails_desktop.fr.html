Barre de navigation supérieure

En haut à gauche de l'écran, vous pouvez voir deux menus :

  • le menu Applications
  • le menu Emplacements

Menu des applications

Le menu Applications rassemble des raccourcis vers les logiciels inclus dans Tails et vers les utilitaires GNOME.

Aide : pour accéder au Manuel de GNOME, choisissez Utilitaires ▸ Aide
Paramètres : pour modifier les paramètres tels que le clavier, la souris, le pavé tactile, ou les paramètres d'affichage, choisissez Outils système ▸ Paramètres

Pour apprendre les différents raccourcis clavier de GNOME, cliquez sur Clavier depuis les Paramètres puis ouvrez l'onglet Raccourcis.

Par défaut, les personnalisations de ce type sont annulées lors de l'extinction de Tails. Consultez la documentation relative à la persistance pour savoir quelles configurations peuvent être conservées de manière persistante entre deux sessions de travail.
Seahorse : pour gérer vos clés OpenPGP, choisissez Utilitaires ▸ Mots de passe et clés

Sous-menu Favoris

Dans le sous-menu Favoris, quelques raccourcis permettent d'accéder à des applications fréquemment utilisées :

Navigateur Tor : navigateur Web
Voir la documentation correspondante
Thunderbird : client email
Voir la documentation correspondante
Pidgin : client de messagerie instantanée
Voir la documentation correspondante
KeePassX : gestionnaire de mots de passe
Voir la documentation correspondante

Terminal GNOME : pour utiliser la ligne de commande

Menu Emplacements

Le menu Emplacements fournit un accès direct à différents dossiers et supports de stockage.

Statut et circuits de Tor

L'icône de statut de Tor et Onion Circuits vous permettent de voir l'état de Tor. Voir la documentation correspondante.

Applet OpenPGP

Avec l'applet OpenPGP vous pouvez chiffrer et déchiffrer le presse-papier en utilisant OpenPGP. Voir la documentation correspondante.

Accès universel

Le menu d'accès universel vous permet d'activer le lecteur d'écran, le clavier visuel, l'affichage de grand texte et d'autres technologies d'accessibilité. Voir la documentation correspondante.

Dispositions du clavier

Le menu de disposition du clavier vous permet de changer la disposition du clavier et la méthode de saisie pour l'écriture non-latine.

Menu système

Dans le coin à droite de la barre du haut, le menu système vous permet de gérer les paramètres système et votre ordinateur.

Voir également la documentation sur comment se connecter à un réseau.

Verrouiller votre écran

Cliquez sur le bouton dans le menu système pour verrouiller votre écran avec un mot de passe.

  • Si vous avez défini un mot de passe d'administration lors du démarrage de Tails, vous pouvez déverrouiller votre écran avec votre mot de passe d'administration.

  • Dans le cas contraire, vous pouvez définir un mot de passe pour déverrouiller votre écran lorsque vous le verrouillez pour la première fois.

Vue d'ensemble des activités

Pour accéder à vos fenêtres et applications vous pouvez au choix :

  • Choisir Applications ▸ Vue d'ensemble des activités.
  • Amener le curseur de la souris dans le coin en haut à gauche.
  • Appuyer sur la touche fenêtre de votre clavier.

Vous pouvez voir les fenêtres et applications dans la vue d'ensemble. Vous pouvez également faire une recherche parmis vos applications, fichiers et dossiers.

Raccourcis sur le bureau

Dossier personnel : raccourci vers le dossier de l'utilisateur par défaut
Documentation de Tails : ouvre une copie locale du site web et de la documentation de Tails
Signaler une erreur : vous aide à dépanner Tails
Corbeille : où sont déplacés les fichiers "effacés"

Gérer les fichiers avec Nautilus

Nautilus est le gestionnaire de fichiers de GNOME, un client FTP, SFTP et plus encore.

Pour gérer les fichiers locaux, suivez les liens sur le bureau ou depuis le menu Emplacements dans le coin supérieur gauche de l'écran. Pour déplacer des fichiers ou dossiers, vous pouvez les faire glisser d'une fenêtre à une autre.

Pour vous connecter à un serveur FTP ou SFTP, choisissez Connexion à un serveur.
