Dona

Ci sono diversi modi per arrestare Tails:

  • By clicking on the system menu located in the upper right corner of the screen and choosing either:

    • to shutdown the computer or
    • to restart the computer.

  • By physically removing the USB stick or DVD that Tails is running from.

Questo metodo potrebbe, in rare occasioni, rovinare il file system del tuo volume persistente Usa questo metodo solo in caso di emergenza.

Se non puoi attivare la persistenza dopo aver usato questo metodo, dovresti essere in grado di recuperare molti dei tuoi dati facendo un controllo del file system nel volume persistente.

Quando il sistema si arresta, i dati memorizzati in RAM vengono cancellati per proteggerli dall' attacco di avvio freddo.
