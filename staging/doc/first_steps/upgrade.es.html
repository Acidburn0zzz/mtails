Tails incluye un mecanismo para actualizar las memorias USB a la última versión. En algunos casos es imposible hacer una actualización automática y deberás hacer una actualización manual. Esta página describe ambas técnicas.

Nuestras actualizaciones corrigen problemas de seguridad importantes, por lo que deberías actualizar tan pronto como te sea posible.

Ambos métodos solo funcionan si la memoria USB actualizada fue instalada con el Tails installer. El almacenamiento persistente en el dispositivo será preservado.

Si utilizas Tails desde un DVD, necesitas grabar un nuevo DVD.

Si utilizas Tails desde una memoria USB instalada desde macOS utilizando la linea de comandos, necesitas reinstalar.

  1. Actualización automática usando Tails Upgrader
  2. Actualización manual usando Tails Installer

Actualización automática usando Tails Upgrader

Después de iniciar Tails y conectarse a Tor, el Actualizador de Tails automáticamente comprueba si hay actualizaciones disponibles y luego propone que actualices el dispositivo. Las actualizaciones son comprobadas y descargadas a través de Tor.

Las ventajas de esta técnica son las siguientes:

  • Sólo necesitas una memoria USB de Tails. La actualización se hace sobre la marcha desde un Tails en ejecución. Después de actualizar, puedes reiniciar y utilizar la nueva versión.
  • La actualización es una descarga mucho más pequeña que una imagen ISO completa.
  • El mecanismo de actualización incluye la verificación criptográfica de la misma. Ya no tienes que verificar la imagen ISO por ti mismo.

Requerimientos:

  • Un dispositivo Tails, instalado utilizando el Tails Installer.
  • Una conexión de Internet.

Después de conectarte a Tor si hay una actualización disponible un cuadro de diálogo aparece y te propone actualizar tu dispositivo.

  • Se recomienda cerrar todas las aplicaciones abiertas durante la actualización.
  • Descargar la actualización puede llevar mucho tiempo, desde algunos minutos a varias horas.
  • La red será deshabilitada después de descargar la actualización.

Si decides realizar la actualización, haz click en Actualizar ahora, y sigue las instrucciones a través del proceso de actualización.

Si te has perdido una actualización, cada actualización se instalará una tras otra. Por ejemplo, si tienes Tails 1.3 y la versión actual es 1.3.2, primero se instalará la actualización a 1.3.1 y después de reiniciar Tails, se instalará la actualización a 1.3.2.

Si no puedes actualizar al inicio (por ejemplo, si no tienes conexión de internet en ese momento), puedes iniciar Tails Upgrader más tarde abriendo una Terminal y ejecutando el siguiente comando:

tails-upgrade-frontend-wrapper

Te recomendamos que leas las notas de lanzamiento de la última versión, ya que documentan todos los cambios de esta nueva versión:

  • nuevas funcionalidades
  • problemas resueltos
  • problemas conocidos que se han identificado

Puede que también contengan instrucciones para actualizar.

Si hay un error el asistente te propondrá leer una de las siguientes páginas:

  • Si ocurre algún error mientras compruebas por las actualizaciones disponibles:
    file:///usr/share/doc/tails/website/doc/upgrade/error/check.es.html
  • Si algún error ocurre mientras descargas la actualización:
    file:///usr/share/doc/tails/website/doc/upgrade/error/download.es.html
  • Si algún error ocurre mientras instalas la actualización:
    file:///usr/share/doc/tails/website/doc/upgrade/error/install.es.html

Actualización manual usando Tails Installer

Te recomendamos que leas las notas de lanzamiento de la última versión, ya que documentan todos los cambios de esta nueva versión:

  • nuevas funcionalidades
  • problemas resueltos
  • problemas conocidos que se han identificado

Puede que también contengan instrucciones para actualizar.

Puede que no siempre sea posible hacer una actualización automática como se describe anteriormente. Por ejemplo, cuando:

  • No hay ninguna actualización automática disponible en nuestro sitio web para esta versión.
  • La actualización automática es imposible por razones técnicas (no hay suficiente memoria, no hay suficiente espacio libre en el dispositivo, etc.).
  • Deseas actualizar desde otro dispositivo Tails que ya tiene instalada una versión más reciente, por ejemplo al trabajar sin conexión.
  • La actualización automática falló y necesitas reparar tu dispositivo Tails.

Después de conectar a Tor, un diálogo te informará si debes actualizar tu memoria USB con Tails Installer a una versión más reciente de Tails. Para hacerlo sigue nuestras instrucciones de actualización manual.
