Spenden

Es gibt mehrere Möglichkeiten, Tails herunterzufahren:

  • Indem Sie das Systemmenü, welches in der oberen rechten Ecke des Bildschirm ist, anklicken und Sie entweder

    • auswählen, um den Computer herunterzufahren oder
    • auswählen, um den Computer neuzustarten.

  • Indem Sie den USB-Stick oder die DVD, von der Tails läuft, physisch entfernen.

Diese Methode kann in einigen raren Fällen das Dateisystem Ihres beständigen Datenträgers beschädigen. Benutzen Sie diese Methode nur im Notfall.

Sollten Sie den beständigen Datenträger nicht mehr aktivieren können, nachdem Sie diese Methode benutzt haben, sollten Sie den Großteil Ihrer Daten wiederherstellen können, indem Sie das Dateisystem des beständigen Datenträgers überprüfen.

Während des Herunterfahrens werden die Daten, die im RAM gespeichert sind, gelöscht, um Kaltstartattacken zu verhindern.
