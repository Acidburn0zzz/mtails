There are several ways of shutting down Tails:

  • By clicking on the system menu located in the upper right corner of the screen and choosing either:

    • to shutdown the computer or
    • to restart the computer.

  • By physically removing the USB stick or DVD that Tails is running from.

This method might in rare occasions break the file system of your persistent volume. Use this method only in case of emergency.

If you cannot enable persistence after using this method, you should be able to recover most of your data by doing a file system check on the persistent volume.

While shutting down, the data stored in RAM is erased to protect from cold boot attacks.
