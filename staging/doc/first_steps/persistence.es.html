Donar

Si inicias Tails desde un dispositivo USB puedes crear un volumen persistente en el espacio libre restante del dispositivo con Tails Installer. Los archivos en el volumen persistente se guardan encriptados y permanecen disponibles entre sesiones de trabajo separadas.

Sólo es posible crear un volumen persistente si la memoria USB ha sido instalada usando Tails Installer.

Esto requiere de una memoria USB de al menos 8 GB.

You can use this persistent volume to store any of the following:

  • Personal files
  • Some settings
  • Additional software
  • Encryption keys

El volumen persistente es una partición cifrada protegida por una frase contraseña.

Una vez que crees el volumen persistente, puedes escoger activarlo o no cada vez que inicias Tails.

El uso de un volumen persistente en un sistema diseñado para proveer anonimidad y no dejar trazas es un tema complicado.
