Tails y el escritorio de GNOME incluyen tecnologías de asistencia para apoyar a los usuarios con diversos impedimentos y necesidades especiales e interactuar con dispositivos de asistencia comunes.

  1. Menú de acceso universal
  2. Atajos de teclado
  3. Lector de pantalla
  4. Teclado en pantalla
  5. Dasher aplicación gráfica de entrada de texto

Menú de acceso universal

Puedes activar la mayoría de estas tecnologías desde el menú de acceso universal (el ícono que se asemeja a una persona) en la barra superior:

Estas tecnologías son documentadas en más detalle en la [documentación GNOME para el acceso universal(https://help.gnome.org/users/gnome-help/stable/a11y.html.es).

Atajos de teclado

Turn screen reader on or offAlt+Super+S
Turn zoom on or offAlt+Super+8

To access the universal access menu using the keyboard:

  1. Press Ctrl+Alt+Tab to move the keyboard focus to the top bar.

  2. Use the arrow keys on the keyboard to select the universal access menu.

  3. Press Enter to open the menu.

  4. Use the up and down arrow keys to select items in the menu.

  5. Press Space to toggle the selected item.

  6. Press Esc to close the menu.

Lector de pantalla

Para que el ordenador diga los elementos de la pantalla puedes activar la función de accesibilidad Lector de Pantalla.

Para más información detallada revisa la documentación del Lector de Pantalla Orca documentation.

The screen reader takes around 10 seconds to start.

El lector de pantalla no funciona con el Navegador Inseguro.

Teclado en pantalla

Si prefieres un puntero sobre el teclado puedes activar la función de accesibilidad Teclado en Pantalla.

Para más información detallada revisa la documentación oficial del Teclado en Pantalla GNOME.

Dasher aplicación gráfica de entrada de texto

Si operas la computadora con una sola mano (por joystick, pantalla táctil, o ratón) o sin manos (por cabeza-ratón o eyetracker), puedes utilizar el Dasher aplicación gráfica de entrada de texto predictivo.

Dasher está disponible desde Aplicaciones ▸ Acceso Universal ▸ Dasher.
