Pour désinstaller Tails d'une clé USB et l'utiliser pour autre chose, vous devez la réinitialiser.

Le contenu de la clé USB sera perdu lors de l'opération, mais un attaquant sera quand même capable de dire que Tails fut installé sur cette clé USB en utilisant des techniques de restauration de données à moins que vous n'effaciez de manière sécurisée la totalité de l'espace disque disponible après coup.

Les instructions suivantes ne marchent pas sur Windows XP.
La version de Diskpart sur Windows XP ne liste pas les périphériques amovibles.

En utilisant Diskpart

Vous pourriez écraser n'importe quel disque dur sur l'ordinateur.
Si à un moment vous n'êtes pas sûr du choix du périphérique, arrêtez-vous là.

  1. Assurez-vous que la clé USB que vous voulez réinitialiser est débranchée.

  2. Cliquez sur le bouton Démarrer, et choisir Tous les programmes ▸ Accessoires ▸ Invite de commandes, pour ouvrir l'interpréteur de commande,

    Plus d'aide sur comment lancer l'interpréteur de commande (en anglais)

  3. Exécutez la commande diskpart pour lancer Diskpart.

  4. Exécutez la commande list disk pour obtenir des informations sur chaque disque de l'ordinateur.

    Par exemple :

    Diskpart> list disk

  Disk ###  Status      Size     Free     Dyn  Gpt
  --------  ----------  -------  -------  ---  ---
  Disk 0    Online        80 GB      0 B

  5. Branchez la clé USB que vous voulez réinitialiser. Relancez la commande list disk.

    Un nouveau disque, correspondant à cette clé USB, apparaît dans la liste.

    Par exemple :

     Diskpart> list disk

   Disk ###  Status      Size     Free     Dyn  Gpt
   --------  ----------  -------  -------  ---  ---
   Disk 0    Online        80 GB      0 B
   Disk 1    Online         8 GB      0 B

    Assurez-vous que la taille correspond à la taille de la clé USB que vous voulez réinitialiser. Notez le numéro de disque assigné par Diskpart à la clé USB.

  6. Pour sélectionner la clé USB, exécutez la commande suivante : select disk=numéro. Remplacez numéro par le numéro de disque de la clé USB que vous voulez réinitialiser.

  7. Exécutez la commande clean pour supprimer la table de partition de la clé USB.

  8. Exécutez la commande convert mbr pour créer une table de partition sur la clé USB.

  9. Exécutez la commande create partition primary pour créer une partition primaire sur la clé USB.

Dépannage

Voir la documentation sur Diskpart de l'assistance Microsoft.
