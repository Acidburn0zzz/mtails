Para desinstalar Tails de una memoria USB y usarla para otra cosa, la debes resetear.

El contenido de la memoria USB se perderá durante esta operación, pero un atacante podría aún ver que Tails se instaló en ese dispositivo usando técnicas de recuperación de datos a menos que limpies de modo seguro todo el espacio disponible en la memoria USB después.

Podrías sobreescribir cualquier disco de tu ordenador. Si no estás seguro del dispositivo a elegir, mejor para.

  1. Asegúrate de que el dispositivo USB que quieres resetear está desconectado.

  2. Iniciar Disk Utility.

    Una lista de todos los dispositivos de almacenamiento del ordenador aparece en el panel izquierdo de la ventana.

  3. Conecta el dispositivo USB que quieres reiniciar.

    Un nuevo dispositivo aparecerá en la lista de dispositivos de almacenamiento. Este nuevo dispositivo corresponde al dispositivo USB que has enchufado. Haz click en él.

  4. En la lista de dispositivos de almacenamiento, verifica que la memoria USB corresponde con el que deseas reiniciar, su marca, su tamaño, etc.

  5. Para reiniciar el dispositivo:

    1. Si hay una pestaña de Borrar en el panel derecho, haz click en ella y luego en el botón Borrar abajo del todo a la derecha.

    2. Si hay un botón Borrar en la parte media superior, haz click en el.

  6. En el dialogo de confirmación, haz click en el botón Borrar para confirmar.

Si falla el borrado, intenta formatear la memoria USB usando la línea de comandos primero.

Solución de problemas

Formatear usando la línea de comandos

  1. Desconecta la memoria USB del ordenador.

  2. Abre Terminal desde Applications ▸ Utilities ▸ Terminal.app.

  3. Ejecuta el siguiente comando:

    diskutil list

    Devuelve una lista de los dispositivos de almacenamiento del sistema. Por ejemplo:

    $ diskutil list
/dev/disk0
  #:                     TYPE NAME         SIZE      IDENTIFIER
  0:    GUID_partition_scheme             *500.1 GB  disk0
  1:                      EFI              209.7 MB  disk0s1
  2:                Apple_HFS MacDrive     250.0 GB  disk0s2
  3:                      EFI              134.1 GB  disk0s3
  4:     Microsoft Basic Data BOOTCAMP     115.5 GB  disk0s4

  4. Inserta la memoria USB en el ordenador.

  5. Ejecuta otra vez el mismo comando:

    diskutil list

    Tu memoria USB debería aparecer como un nuevo dispositivo en la lista. Verifica que su tamaño corresponde al tamaño de tu memoria USB.

    $ diskutil list
/dev/disk0
  #:                     TYPE NAME         SIZE      IDENTIFIER
  0:    GUID_partition_scheme             *500.1 GB  disk0
  1:                      EFI              209.7 MB  disk0s1
  2:                Apple_HFS MacDrive     250.0 GB  disk0s2
  3:                      EFI              134.1 GB  disk0s3
  4:     Microsoft Basic Data BOOTCAMP     115.5 GB  disk0s4
/dev/disk1
  #:                     TYPE NAME         SIZE      IDENTIFIER
  0:   FDisk_partition_scheme             *8.0 GB    disk1
  1:                Apple_HFS Untitled 1   8.0 GB    disk1s1

  6. Toma nota del nombre de dispositivo de tu memoria USB. En este ejemplo, la memoria USB es de 8.0 GB y su nombre de dispositivo es /dev/disk1. El tuyo puede ser diferente.

    Si no estás seguro del nombre del dispositivo deberías parar ahora o te arriesgas a sobreescribir cualquier otro disco duro del sistema.

  7. Ejectura la siguiente orden para remover con seguridad la memoria USB. Reemplaza [dispositivo] con el nombre del dispositivo encontrado en el paso 6.

    diskutil unmountDisk [dispositivo]

  8. Ejecuta el siguiente comando para formatear la memoria USB. Reemplaza [dispositivo] con el nombre del dispositivo encontrado en el paso 6.

    sudo fdisk -iy [dispositivo]

  9. Puedes probar reiniciar la memoria USB de nuevo usando Disk Utility desde el inicio de esta pagina.
