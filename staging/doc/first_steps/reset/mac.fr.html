Faire un don

Pour désinstaller Tails d'une clé USB et l'utiliser pour autre chose, vous devez la réinitialiser.

Le contenu de la clé USB sera perdu lors de l'opération, mais un attaquant sera quand même capable de dire que Tails fut installé sur cette clé USB en utilisant des techniques de restauration de données à moins que vous n'effaciez de manière sécurisée la totalité de l'espace disque disponible après coup.

Vous pourriez écraser n'importe quel disque dur sur l'ordinateur. Si à un moment vous n'êtes pas sûr du choix du périphérique, arrêtez-vous là.

  1. Assurez-vous que la clé USB que vous voulez réinitialiser est débranchée.

  2. Démarrez l'Utilitaire de disque.

    Une liste de tous les périphériques de stockage de l'ordinateur apparaît dans le panneau gauche de la fenêtre.

  3. Branchez la clé USB que vous voulez réinitialiser.

    Un nouveau périphérique apparaît dans le liste des périphériques de stockage. Ce nouveau périphérique correspond à la clé USB que vous venez de brancher. Cliquez dessus.

  4. Dans la liste des périphériques de stockage, vérifiez que le périphérique correspond à la clé USB que vous voulez réinitialiser, sa marque, sa taille, etc.

  5. Pour réinitialiser la clé USB :

    1. Si il y a un onglet Effacer dans le panneau droit, cliquez dessus puis sur le bouton Effacer en bas à droite.

    2. Si il y a un bouton Effacer en haut au milieu, cliquez dessus.

  6. Dans la boite de confirmation, cliquez sur le bouton Effacer pour confirmer.

Si le processus d'effacement échoue, essayez de formater la clé USB en ligne de commande d'abord.

Résolution des problèmes

Formater en utilisant la ligne de commande

  1. Débranchez la clé USB de votre ordinateur.

  2. Ouvrir Terminal depuis Applications ▸ Utilitaires ▸ Terminal.app.

  3. Exécuter la commande suivante :

    diskutil list

    Elle retourne une liste des périphériques de stockage sur votre système. Par exemple :

    $ diskutil list
/dev/disk0
  #:                     TYPE NAME         SIZE      IDENTIFIER
  0:    GUID_partition_scheme             *500.1 GB  disk0
  1:                      EFI              209.7 MB  disk0s1
  2:                Apple_HFS MacDrive     250.0 GB  disk0s2
  3:                      EFI              134.1 GB  disk0s3
  4:     Microsoft Basic Data BOOTCAMP     115.5 GB  disk0s4

  4. Branchez la clé USB dans votre ordinateur.

  5. Exécutez à nouveau la même commande :

    diskutil list

    Votre clé USB apparaît comme un nouveau périphérique dans la liste. Vérifiez si sa taille correspond à la taille de votre clé USB.

    $ diskutil list
/dev/disk0
  #:                     TYPE NAME         SIZE      IDENTIFIER
  0:    GUID_partition_scheme             *500.1 GB  disk0
  1:                      EFI              209.7 MB  disk0s1
  2:                Apple_HFS MacDrive     250.0 GB  disk0s2
  3:                      EFI              134.1 GB  disk0s3
  4:     Microsoft Basic Data BOOTCAMP     115.5 GB  disk0s4
/dev/disk1
  #:                     TYPE NAME         SIZE      IDENTIFIER
  0:   FDisk_partition_scheme             *8.0 GB    disk1
  1:                Apple_HFS Untitled 1   8.0 GB    disk1s1

  6. Notez le nom de périphérique de votre clé USB. Dans cet exemple, la clé USB a 8.0 GB et son nom de périphérique est /dev/disk1. Le nom de la votre peut être différent.

    Si vous n'êtes pas sûr du nom de périphérique, vous devriez arrêter la procédure ou vous risquez d'écraser les données d'un disque dur de votre système.

  7. Exécutez la commande suivante pour faire un effacement sécurisé de la clé USB. Remplacez [device] par le nom de périphérique trouvé à l'étape 6.

    diskutil unmountDisk [device]

  8. Exécutez la commande suivante pour formater la clé USB. Remplacez [device] par le nom de périphérique trouvé à l'étape

    1. sudo fdisk -iy [device]

  9. Vous pouvez essayer à nouveau de réinitialiser la clé USB avec l'Utilitaire de disque en retournant au début de cette page.
