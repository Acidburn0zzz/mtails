Pour désinstaller Tails d'une clé USB et l'utiliser pour autre chose, vous devez la réinitialiser.

Le contenu de la clé USB sera perdu lors de l'opération, mais un attaquant sera quand même capable de dire que Tails fut installé sur cette clé USB en utilisant des techniques de restauration de données à moins que vous n'effaciez de manière sécurisée la totalité de l'espace disque disponible après coup.

  1. En utilisant GNOME Disques
  2. Réinitialiser une clé USB Tails depuis elle-même

En utilisant GNOME Disques

Vous pourriez écraser n'importe quel disque dur sur l'ordinateur. Si à un moment vous n'êtes pas sûr du choix du périphérique, arrêtez-vous là.

  1. Assurez-vous que la clé USB que vous voulez réinitialiser est débranchée.

  2. Choisir Applications ▸ Utilitaires ▸ Disques pour lancer GNOME Disques.

    Une liste de tous les périphériques de stockage de l'ordinateur apparaît dans le panneau gauche de la fenêtre.

  3. Branchez la clé USB que vous voulez réinitialiser.

    Un nouveau périphérique apparaît dans le liste des périphériques de stockage. Ce nouveau périphérique correspond à la clé USB que vous venez de brancher. Cliquez dessus.

  4. Dans la partie droite de la fenêtre, vérifiez que le périphérique correspond à la clé USB que vous voulez réinitialiser, sa marque, sa taille, etc.

  5. Pour réinitialiser la clé USB, cliquez sur le bouton dans la barre de titre puis sur Formater le disque….

  6. Dans la boîte de dialogue Formater le disque :

    1. Si vous souhaitez effacer les données de manière sécurisée, choisissez Écraser les données existante avec des zéros dans le menu déroulant Effacer.

    2. Sélectionnez Compatible avec tout les systèmes et périphériques (MBR/DOS) dans le menu déroulant Partitionnement.

    Cliquez alors sur le bouton Formater....

  7. Dans le boîte de dialogue de confirmation, cliquez sur le bouton Formater pour confirmer.

  8. Pour vous assurer que tous les changements ont été écrits sur la clé USB, cliquez sur le bouton dans la barre de titre.

Réinitialiser une clé USB Tails depuis elle-même

Si Tails est votre seul système Linux, vous pouvez généralement réinitialiser une clé USB Tails directement depuis celle-ci, en cours d'exécution.

  1. Au démarrage de Tails, ajoutez toram comme option de démarrage dans le menu du chargeur d’amorçage. Pour des instructions détaillées, voir la documentation sur l'utilisation du menu du chargeur d’amorçage.

  2. Si Tails démarre comme d'habitude, suivre les instructions pour réinitialiser une clé USB en utilisant GNOME Disques.

    Si le système n'arrive pas à démarrer, cela signifie que l'ordinateur n'a pas assez de mémoire pour ce mode opératoire. Essayez avec un autre ordinateur, ou trouvez un autre système Linux, comme une autre clé USB Tails, pour effectuer la réinitialisation.
