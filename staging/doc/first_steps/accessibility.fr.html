Tails et le bureau GNOME incluent des technologies d'assistance pour aider les personnes avec divers handicaps et des besoins spécifiques et pour interagir avec des périphériques d'assistance courants.

  1. Menu d'accès universel
  2. Raccourcis clavier
  3. Lecteur d'écran
  4. Clavier visuel
  5. Dasher, une application de saisie prédictive de texte

Menu d'accès universel

Vous pouvez activer la plupart de ces technologies depuis le menu d'accès universel (l'icône qui ressemble à une personne) dans la barre du haut :

Ces technologies sont documentées plus en détail dans la documentation GNOME pour l'accès universel.

Raccourcis clavier

Activer ou désactiver le lecteur d'écranAlt+Super+S
Activer ou désactiver le zoomAlt+Super+8

Pour accéder au menu d'accès universel en utilisant le clavier :

  1. Appuyez sur Ctrl+Alt+Tab pour déplacer le focus du clavier vers la barre supérieure.

  2. Utilisez les touches directionnelles du clavier pour sélectionner le menu d'accès universel.

  3. Appuyez sur Entrée pour ouvrir le menu.

  4. Utilisez les touches directionnels haut et bas pour sélectionner des éléments dans le menu.

  5. Appuyez sur Espace pour activer ou désactiver l'élément sélectionné.

  6. Appuyez sur Échap pour fermer le menu.

Lecteur d'écran

Afin d'obtenir une lecture des éléments présents à l'écran, vous pouvez activer la fonctionnalité d'accessibilité Lecteur d'écran.

Pour plus d'informations, veuillez consulter la documentation officielle du lecteur d'écran Orca .

Le lecteur d'écran met environ 10 secondes pour démarrer.

Le lecteur d'écran ne marche pas avec le Navigateur web non-sécurisé.

Clavier visuel

Si vous préférez utiliser un dispositif de pointage pour le clavier, vous pouvez activer la fonctionnalité d'accessibilité Clavier visuel.

Pour une documentation plus détaillée, consultez la documentation officielle du clavier visuel de GNOME.

Dasher, une application de saisie prédictive de texte

Si vous utilisez un ordinateur avec une seule main (avec un joystick, un écran tactile, ou une souris) ou sans les mains (souris guidé par les mouvements de la tête ou un suivi de l'œil), vous pouvez utiliser l'application graphique de saisie de texte prédictive Dasher.

Dasher est disponible depuis Applications ▸ Accès universel ▸ Dasher.
