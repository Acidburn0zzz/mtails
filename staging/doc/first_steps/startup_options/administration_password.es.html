Donar

En Tails se necesita una contraseña de administración (también llamada contraseña de root) para realizar tareas de administración del sistema.
Por ejemplo:

  • Para instalar software adicional
  • Para acceder al disco duro interno de la computadora
  • Para ejecutar comandos con sudo

Por defecto, la contraseña de administración está deshabilitada por mayor seguridad. Esto puede prevenir que un atacante con acceso físico o remoto a tu sistema Tails adquiera privilegios de administración y realice tareas administrativas contra tu voluntad.

Habilita una contraseña de administración

Para poder realizar tareas administrativas, necesitas configurar una contraseña de administrador cuando inicies Tails, usando el Tails Greeter.

  1. Cuando aparezca el Tails Greeter, haz click en el botón .

  2. Cuando aparezca el cuadro de diálogo Configuración adicional, da click en Contraseña de administrador.

  3. Elige una contraseña de tu elección en las cajas de texto Contraseña y Verificar Contraseña, y luego haz click en Añadir.

Como abrir una terminal de root

Para abrir una terminal de root durante tu sesión activa, puedes realizar alguna de las opciones siguientes:

  • Choose Aplicaciones ▸ Herramientas del sistema ▸ Terminal de Root.

  • Ejecuta sudo -i en una terminal.
