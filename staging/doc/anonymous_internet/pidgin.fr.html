Faire un don

Pour dialoguer et envoyer des messages instantanés, Tails fournit la Messagerie Instantanée Pidgin.

Vous pouvez l'utiliser pour vous connecter à des serveurs IRC ou XMPP (aussi connus sous le nom de Jabber), et utiliser simultanément plusieurs comptes différents.

Pour lancer Pidgin choisir Applications ▸ Internet ▸ Messagerie internet Pidgin ou cliquer sur l'icône Pidgin dans le sous-menu Favoris.

Pour de la documentation plus détaillée, référez-vous au manuel utilisateur officiel de Pidgin (en anglais).

Comptes pré-définis

Un compte est configuré par défaut dans Pidgin :

  • irc.oftc.net pour se connecter au serveur IRC d'OFTC.

Ce compte est désactivé lors du démarrage de Tails. Pour l'activer, choisir Comptes ▸ Activer un compte et sélectionnez dans le sous-menu le compte que vous voulez activer.

Chiffrement Off-the-record (OTR)

Comme expliqué sur sa page officielle, la messagerie Off-the-Record vous permet d'avoir des conversations privées via messagerie instantanée en fournissant :

  • Chiffrement
    Personne d'autre ne peut lire vos messages.
  • Authentification
    Vous avez la garantie que votre correspondant est la personne que vous croyez.
  • Déni possible
    Les messages que vous envoyez n'ont pas de signature numérique vérifiable par une tierce personne. N'importe qui peut falsifier des messages après la conversation pour faire comme s'ils venaient de vous. Cependant, pendant une conversation, votre correspondant est assuré que les messages qu'il reçoit sont authentiques et intacts.
  • Confidentialité persistante
    Si vous perdez le contrôle de vos clés privées, aucune conversation précédente ne sera compromise.

OTR est désactivé par défaut, et vos conversations ne sont pas privées.

Les fichiers transférés ne sont pas chiffrés par OTR. OTR chiffre seulement les conversations.

Pour conserver vos clés OTR et vos préférences à travers plusieurs sessions de travail, vous pouvez activer l'option de persistance Pidgin.
Lors d'une conversation privée OTR via IRC, un message envoyé en utilisant la commande /me n'est pas chiffré. La personne le recevant en est informée par un avertissement.

Serveurs IRC bloquant Tor

Certains serveurs IRC bloquent les connexions en provenance de Tor car Tor est parfois utilisé pour envoyer du spam.

  • Accéder à Freenode via Tor (le service caché mentionné sur cette page est souvent indisponible) (en anglais)
  • OFTC et Tor (en anglais)
  • Liste de réseau IRC/de chat bloquant ou supportant Tor (en anglais)

Génération aléatoire du nom d'utilisateur

À chaque démarrage de Tails, un nom d'utilisateur aléatoire est généré pour tous les comptes Pidgin.

Le générateur utilise une liste de prénoms anglais communs, et les modifie, de telle sorte qu'ils deviennent quasiment uniques, et ne révèlent pas que vous utilisez Tails.

Il est basé sur le mixer de langues de Christopher Pound.

Si vous voulez utiliser le même nom d'utilisateur pour plusieurs sessions de travail, vous pouvez activer l'option de persistance Pidgin.

Ajouter la prise en charge d'un autre protocole

Pour des raisons de sécurité, il est uniquement possible d'utiliser Pidgin avec les protocoles IRC et XMPP dans Tails. Voici les prérequis pour activer d'autres protocoles pris en charge par Pidgin par ailleurs :

  1. La prise en charge dans Pidgin de ce protocole doit avoir été testée avec succès dans Tails.
  2. Quelqu'un doit se porter volontaire pour maintenir sur le long terme sa prise en charge dans Tails.
  3. Quelqu'un doit vérifier que l'historique de sécurité du plugin désiré (incluant les bugs en cours) est assez bon.

Si vous voulez travaillez sur ce problème, voir notre page participer.
