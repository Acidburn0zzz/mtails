Impossible de déterminer si une mise à jour est disponible depuis le site web.

Ceci est probablement dû à un problème de connexion réseau.

Vérifiez votre connexion réseau et redémarrez Tails pour tenter une nouvelle mise à jour.

Si le problème persiste, signalez cette erreur et incluez dans votre rapport les informations de débogage qui apparaissent dans le message d'erreur.

Cette erreur peut également être causée par :

  • une erreur de Tails Upgrader
  • une attaque ou un échec de la part du nœud de sortie Tor utilisé, de votre fournisseur d'accès à Internet, ou de notre site web.

Nous envoyer un rapport est le meilleur moyen d'analyser plus avant l'erreur.
