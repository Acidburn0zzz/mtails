Donar

Tails es Software Libre: puedes descargarlo, usarlo y compartirlo sin restricciones.

El código fuente de Tails se lanza bajo la licencia GNU/GPL (version 3 or above) y su Copyright (C) pertenece a Tails developers tails@boum.org. Cualquier excepción a esta regla está documentada aquí abajo o en el archivo de código afectado.

Sin embargo, Tails incluye firmware no-libre para así poder funcionar en la mayor cantidad de hardware posible.

Código fuente

La mayoría del software incluido en Tails es tomado directamente de paquetes Debian, y no es modificado ni recompilado por Tails.

  • El código fuente del software específico de Tails está disponible en [nuestros repositorios Git|contribute/git]].

  • El código fuente de los paquetes de Debian incluídos en Tails está disponible en el snapshot APT que creamos para esta versión de Tails.
