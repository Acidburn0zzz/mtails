Tails é baseado no Debian 9 (Stretch).

Programas inclusos

  • GNOME, um ambiente desktop atrativo e intuitivo (Mais...) Copy →↵…«»“”-–—

Rede

  • Tor with:
    • stream isolation
    • regular, obfs2, obfs3, obfs4, and ScrambleSuit bridges support
    • the Onion Circuits graphical frontend (More...)
  • NetworkManager for easy network configuration (More...)
  • Tor Browser (More...), a web browser based on Mozilla Firefox and modified to protect your anonymity with:
    • Torbutton for anonymity and protection against evil JavaScript
    • all cookies are treated as session cookies by default;
    • HTTPS Everywhere transparently enables SSL-encrypted connections to a great number of major websites
    • NoScript to have even more control over JavaScript.
    • uBlock Origin to remove advertisements.
  • Pidgin preconfigured with OTR for Off-the-Record Messaging (More...)
  • OnionShare for anonymous filesharing
  • Thunderbird email client with Enigmail for OpenPGP support, also an RSS and Atom news feed reader (More...)
  • Gobby for collaborative text writing
  • Aircrack-ng for wireless network auditing
  • Electrum, an easy-to-use bitcoin client (More...)

Edição de Desktop

  • LibreOffice.org (Mais...)
  • Gimp e Inkscape para a edição de imagens (Mais...)
  • Scribus para diagramação
  • Gravador de Som GNOME para gravar som (Mais...)
  • Audacity para gravação e edição de áudio (Mais...)
  • PiTIVi para edição não-linear de áudio e vídeo (Mais...)
  • Poedit para editar arquivos .po
  • Simple Scaney SANE para suporte a scanners
  • Brasero para gravar CD/DVDs
  • Sound Juicer para extrair CDs de áudio
  • Traverso um gravador e editor de áudio multipistas
  • BookletImposer para converter documentos PDF lineares em livretos, e vice-versa

Criptografia e privacidade

  • Support for both LUKS and VeraCrypt encrypted volumes (like USB sticks) (More...)
  • GnuPG, a implementação GNU do OpenPGP para email, criptografia e assinaturas digitais
  • Monkeysign, uma ferramenta para assinatura e troca de chaves OpenPGP
  • PWGen, um gerador de senhas fortes
  • Shamir's Secret Sharing configurado com gfshare e ssss
  • GNOME screen keyboard for accessibility, and as a countermeasure against hardware keyloggers (More...)
  • MAT para anonimizar metadados de arquivos
  • pdf-redact-tools para ajudar a redigir e remover metadados de documentos com segurança antes da publicação
  • KeePassX, gerenciador de senhas (Mais...)
  • GtkHash para calcular somas de verificação (Mais...)
  • Keyringer, uma ferramenta de linha de comando para criptografar segredos compartilhados através de Git (More...)
  • Paperkey ferramenta de linha de comando para fazer cópia de segurança das chaves secretas OpenPGP em papel (Mais...)

A lista completa de pacotes pode ser encontrada no diretório de download de arquivos BitTorrent (arquivos com a extensão .packages).

Características adicionais

Additionally, you can install in Tails any other software available in Debian.

Using the Additional Software persistence feature, additional software can even be installed automatically every time you start Tails.

See the corresponding documentation.

Características adicionais

  • mecanismo automático para atualizar um dispositivo USB para novas versões
  • pode ser utilizado como um sistema virtual dentro do VirtualBox (Mais...)
  • adaptar o sistema (adicionar um programa que não está presente, por exemplo) é relativamente fácil: qualquer pessoa pode construir uma versião própia do Amnesic Incognito Live System em cerca de uma hora em um computador moderno
  • Kernel de 64-bits com PAE , NX-bit e suporte a SMP nos hardwares que o provém
  • Algumas funcionalidades básicas de acessibilidade
  • Algum [[isolamento de aplicações | contribute/design/application_isolation]] com AppArmor
  • Para prevenir ataques do tipo cold-boot e várias técnicas de resgate forense de memória, Tails apaga a maior parte da memória quando o computador é desligado e quando a mídia de inicialização é fisicamente removida.

Idiomas

Ao inicializar Tails, você pode escolher entre um grande número de idiomas, incluindo Albanês, Alemão, Árabe, Azeri, Belga, Bokmål, Catalão, Chinês, Coreano, Croata, Dinamarquês, Eslováquio, Esloveno, Espanhol, Finlandês, Francês, Galês, Grego, Inglês, Húngaro, Indonésio, Italiano, Japonês, Khmer, Lituano, Persa, Polonês, Português, Russo, Sérvio, Sueco, Tcheco, Turco, Ucraniano.

O método requisitado para entrada do teclado é automaticamente habilitado.

Alguns destes idiomas não puderam ser amplamente testados porque nós não falamos todas as línguas do mundo. Se você encontrar problemas ao usar um dos idiomas supostamente implementados, não hesite em contar-nos a respeito. Por exemplo, no Tails provavelmente faltam algumas fontes ou sistemas não-latinos de entrada de dados.

Se você quiser fazer com que o uso do Tails seja mais fácil para pessoas que falam seu idioma, consulte as linhas gerais de tradução.
