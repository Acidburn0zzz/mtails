Tails est un logiciel libre : vous pouvez le télécharger, l'utiliser et le partager sans restriction.

Le code source de Tails est distribué sous licence GNU/GPL (version 3 ou supérieure) et est Copyright (C) Tails developers tails@boum.org. Toute exception à cette règle est documentée soit ci-dessous soit dans le fichier source concerné.

Cependant, Tails inclut des micrologiciels non-libres afin de fonctionner avec le plus de matériel possible.

Code source

La plupart des logiciels inclus dans Tails proviennent directement des paquets Debian en amont et ne sont ni modifiés ni recompilés par Tails.

  • Le code source des logiciels spécifiques à Tails est disponible dans notre dépôt Git.

  • Le code source des paquets Debian inclus dans Tails est disponible dans le snapshot APT que nous avons créé pour cette version de Tails.
