En este contexto, el término 'huella digital' se refiere a la especificidad de Tails en el modo en que se comporta en Internet. Esto se puede usar para determinar si un usuario en particular está usando Tails.

Como se explica en nuestra página de advertencia, cuando usas Tails es posible saber que estás usando Tor. Pero Tails intenta dificultar lo más posible la distinción entre usuarios de Tails y otros usuarios de Tor, especialmente los usuarios de Tor Browser fuera de Tails. Si se pudiera determinar si eres un usuario de Tor Browser dentro o fuera de Tails, esto dará más información sobre ti y esto reduce tu anonimato.

Esta sección explica algunos problemas respecto a la huella digital de Tails, y cómo puede usarse para identificarte como usuario de Tails.

Para los sitios web que visitas

Las páginas web que visitas pueden obtener mucha información sobre tu navegador. Esa información puede incluir su nombre y versión, tamaño de ventana, lista de extensiones disponibles, uso horario, fuentes disponibles, etc.

Para que sea más complicado distinguir usuarios de Tails, el Tor Browser en Tails trata de proporcionar la misma información que el Tor Browser en otros sistemas operativos para tener huellas digitales similares.

Tails incluye uBlock Origin que elimina los anuncios de las p. Si un intruso puede determinar que no estás descargando la publicidad incluída en una página web, eso puede ayudar a identificarte como usuario de Tails.

No se tienen especiales cuidados respecto a la huella digital del Navegador Inseguro.

Para tu ISP o quien administre de tu red local

  • Los puentes de Tor (Tor bridges) son casi siempre una buena forma de esconder de un observador local el hecho de que estás conectado a Tor. Si esto es importante para ti, lee nuestra documentación al respecto: bridge mode.

  • Un sistema Tails genera casi exclusivamente actividad de Tor en la red. Por lo general quienes usan Tor Browser en otro sistema operativo también tienen actividad fuera de Tor, bien sea desde otro navegador u otras aplicaciones. Por eso, la proporción de actividad en Tor puede ser usada para determinar si un usuario de Tor Browser está usando o no Tails. Si compartes tu conección de internet con otros usuarios que no usan Tails, es probablemente más difícil para tu ISP determinar si un usuario está generando sólo tráfico Tor, y quizás usando Tails.

  • Tails no usa el mecanismo de registro de entrada de Tor. Con el [mecanismo de registro de entrada] (https://www.torproject.org/docs/faq#EntryGuards) un usuario Tor siempre usa los mismos relays como primeras opciones. Como Tails no almacena ninguna información de Tor entre diferentes sesiones de trabajo, no almacena información de registro de entrada tampoco. Este comportamiento se podría usar para distinguir usuarios de Tails a través de varias sesiones de trabajo.

  • Al iniciarse, Tails sincroniza el reloj del sistema para asegurarse de que está en hora. Si mientras hace esto la hora se configura en fechas muy pasadas o futuras, Tor se apagará y se reiniciará. Este comportamiento puede ser usado para distinguir usuarios de Tails, especialmente si sucede cada vez que se ejecuta Tails.
