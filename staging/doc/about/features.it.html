Tails è basato su Debian 9 (Stretch).

Software incluso

  • GNOME, un intuitivo ed elegante ambiente desktop (Maggiori informazioni...)

Connessioni

  • Tor with:
    • stream isolation
    • regular, obfs2, obfs3, obfs4, and ScrambleSuit bridges support
    • the Onion Circuits graphical frontend (More...)
  • NetworkManager for easy network configuration (More...)
  • Tor Browser (More...), a web browser based on Mozilla Firefox and modified to protect your anonymity with:
    • Torbutton for anonymity and protection against evil JavaScript
    • all cookies are treated as session cookies by default;
    • HTTPS Everywhere transparently enables SSL-encrypted connections to a great number of major websites
    • NoScript to have even more control over JavaScript.
    • uBlock Origin to remove advertisements.
  • Pidgin preconfigured with OTR for Off-the-Record Messaging (More...)
  • OnionShare for anonymous filesharing
  • Thunderbird email client with Enigmail for OpenPGP support, also an RSS and Atom news feed reader (More...)
  • Gobby for collaborative text writing
  • Aircrack-ng for wireless network auditing
  • Electrum, an easy-to-use bitcoin client (More...)

Edizione Desktop

  • LibreOffice (Maggiori informazioni...)
  • Gimp e Inkscape per l'elaborazione delle immagini (Maggiori informazioni...)
  • Scribus, per l'impaginazione
  • GNOME Sound Recorder for recording sound (More...)
  • Audacity per la regitrazione e l'elaborazione del suono (Maggiori informazioni...)
  • PiTiVi per il montaggio audio/video non-lineare (Plus d'information...)
  • Poedit per editare i file .po
  • Simple Scan e SANE per il supporto degli scanner
  • Brasero per masterizzare CD/DVD
  • Sound Juicer per estrarre l'audio dai CD
  • Traverso un registratore multitraccia ed editor
  • BookletImposer to convert linear PDF documents into booklets, and vice-versa

Cifratura e privacy

  • Support for both LUKS and VeraCrypt encrypted volumes (like USB sticks) (More...)
  • GnuPG, l'implementazione GNU di OpenPGP per le email, la cifratura e la firma dei dati
  • Monkeysign, uno strumento per la firma e lo scambio di chiavi OpenPGP
  • PWGen, un generatore di password forti
  • Shamir's Secret Sharing algoritmo di cifratura usato da gfshare e ssss
  • GNOME screen keyboard for accessibility, and as a countermeasure against hardware keyloggers (More...)
  • MAT per rendere anonimi i metadata nei file
  • pdf-redact-tools to help with securely redacting and stripping metadata from documents before publishing
  • KeePassX Gestore delle password (Plus d'information...)
  • GtkHash per calcolare i checksum (Plus d'information...)
  • Keyringer, uno strumento da riga di comando per scambiare segreti attraverso Git(Plus d'information...)
  • Paperkey uno strumento da riga di comando per fare copie di sicurezza delle chiavi OpenPGP su un foglio di carta (Plus d'information...)

The full packages list can be found in the BitTorrent files download directory (look for files with the .packages extension).

Software aggiuntivo

Additionally, you can install in Tails any other software available in Debian.

Using the Additional Software persistence feature, additional software can even be installed automatically every time you start Tails.

See the corresponding documentation.

Caratteristiche aggiuntive

  • automatic mechanism to upgrade a USB stick to newer versions
  • può essere eseguito come un utente virtualizzato VirtualBox (Maggiori informazioni...)
  • la personalizzazione (per esempio per aggiungere un pezzo di software mancante) è relativamente semplice: una persona può compilare un sistema live anonimo in circa un ora su un computer moderno
  • kernel 64-bit PEA attivato con supporto NX-bit e SMP sull'hardware che lo supporta
  • Alcune caratteristiche di base d'accessibilité
  • Alcune applicazioni confinate con AppArmor
  • To prevent cold-boot attacks and various memory forensics, Tails erases most memory on shutdown and when the boot media is physically removed.

Supporto multilingua

Quando fai partire Tails, puoi scegliere tra un gran numero di lingue, incluso l'Arabo, Azerbaijiano, Catalano, Ceco, Gallese, Danese, Tedesco, Greco, Inglese, Spagnolo, Persiano, Finlandese, Francese, Croato, Ungherese, Indonesiano, Italiano, Giapponese, Khmer, Coreano, Lettone, Bokmal, Tedesco, Polacco, Portoghese, Russo, Slovacco, Sloveno, Albanese, Serbo, Svedese, Turco, Ucraino e Cinese.

Il sistema di input della tastiera è abilitato automaticamente.

Alcuni di questi linguaggi non è stato verificato e noi non parliamo tutte le lingue del mondo. Se trovi alcune imprecisioni usando uno dei linguaggi presumibilmente supportati, non esitare a dircelo. Per esempio Tails probabilmente manca di alcuni font non latini o di sistemi di input.

Se vuoi rendere l'uso di Tails più semplice per la tua lingua parlata, vedi le Linee guida per i traduttori.
