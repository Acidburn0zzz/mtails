Spenden

Tails ist Freie Software: Sie können es ohne Einschränkungen herunterladen, benutzen und verbreiten.

Der Quelltext von Tails wird unter der GNU/GPL (Version 3 oder höher) veröffentlicht und ist Copyright (C) Tails developers tails@boum.org. Jegliche Ausnahme dieser Regel ist entweder unten oder in der betreffenden Quelldatei dokumentiert.

Allerdings enthält Tails nicht-freie Firmware um auf soviel Hardware wie möglich laufen zu können.

Quelltext

Der Großteil der in Tails enthaltenen Software wird direkt von Upstream-Debian-Paketen bezogen und wird für Tails weder verändert noch neu kompiliert.

  • Der Quelltext der für Tails spezifischen Software ist in unseren Git Repositories verfügbar.

  • Der Quelltext der in Tails enthaltenen Debian-Pakete ist im APT-Snapshot, welches wir für die jeweilige Version von Tails erstellt haben, verfügbar.
