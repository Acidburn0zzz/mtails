Tails se basa en Debian 9 (Stretch).

Software incluido

  • GNOME, un intuitivo y atractivo entorno de escritorio (More...)

Redes

  • Tor with:
    • stream isolation
    • regular, obfs2, obfs3, obfs4, and ScrambleSuit bridges support
    • the Onion Circuits graphical frontend (More...)
  • NetworkManager for easy network configuration (More...)
  • Tor Browser (More...), a web browser based on Mozilla Firefox and modified to protect your anonymity with:
    • Torbutton for anonymity and protection against evil JavaScript
    • all cookies are treated as session cookies by default;
    • HTTPS Everywhere transparently enables SSL-encrypted connections to a great number of major websites
    • NoScript to have even more control over JavaScript.
    • uBlock Origin to remove advertisements.
  • Pidgin preconfigured with OTR for Off-the-Record Messaging (More...)
  • OnionShare for anonymous filesharing
  • Thunderbird email client with Enigmail for OpenPGP support, also an RSS and Atom news feed reader (More...)
  • Gobby for collaborative text writing
  • Aircrack-ng for wireless network auditing
  • Electrum, an easy-to-use bitcoin client (More...)

Edición de Escritorio

  • LibreOffice (Más...)
  • Gimp e Inkscape para editar imágenes (Más...)
  • Scribus para diseño de página
  • GNOME Sound Recorder para grabar y editar sonido (Más...)
  • Audacity para grabar y editar sonido (Más...)
  • PiTiVi para edición de audio/video no-lineal (Más...)
  • Poedit para editar archivos .po
  • Simple Scan y SANE para utilizar el scanner
  • Brasero para grabar CD/DVDs
  • Sound Juicer para digitalizar CDs
  • Traverso un grabador y editor de audio multi-pista
  • BookletImposer para convertir documentos PDF en folletos, y viceversa

Encriptación y privacidad

  • Support for both LUKS and VeraCrypt encrypted volumes (like USB sticks) (More...)
  • GnuPG, la implementación GNU de OpenPGP para email, firma y encriptación de datos
  • Monkeysign, una herramienta para la firma y el intercambio de llaves OpenPGP
  • PWGen, un sólido generador de contraseñas
  • Esquema de Shamir usando gfshare y ssss
  • GNOME screen keyboard para accesibilidad, y para prevenir el keyloggers (Más...)
  • MAT para anonimizar metadatos en archivos
  • pdf-redact-tools para ayudar a redactar de forma segura y quitar metadatos de los documentos antes de publicarlos
  • KeePassX gestor de contraseñas (More...)
  • GtkHash para calcular sumas de comprobación (More...)
  • Keyringer, una herramienta de línea de comandos para encriptar secretos compartidos a través de Git (Más...)
  • Paperkey una herramienta de línea de comandos para hacer copias de respaldo de llaves secretas OpenPGP en papel (More...)

La lista completa de paquetes puede encontrarse en el directorio de descargas de archivos BitTorrent (busca archivos con la extensión .packages).

Software adicional

Additionally, you can install in Tails any other software available in Debian.

Using the Additional Software persistence feature, additional software can even be installed automatically every time you start Tails.

See the corresponding documentation.

Funcionalidades adicionales

  • mecanismo automático para actualizar una memoria USB a nuevas versiones
  • puede ejecutarse como un invitado virtual dentro VirtualBox (Más...)
  • personalización (por ejemplo, añadir una pieza faltante de software) es relativamente fácil: puedes compilar un TAILS personalizado en alrededor de una hora en un computador de escritorio moderno
  • Kernel para 64-bit PAE habilitado con soporte para NX-bit y SMP en hardware que lo permita
  • Algunas características accessibility básicas
  • Algunas application isolation con AppArmor
  • Para prevenir ataques de arranque en frío (cold-boot attacks) y otros análisis forenses de memoria, Tails borra la mayoría de la memoria al apagarse, o si remueves el dispositivo desde el que se inició.

Apoyo multilingüe

Cuando comienzas Tails, puedes escoger entre una gran cantidad de lenguajes, incluyendo Arábico, Azerbaiyano, Catalán, Checo, Galés, Danés, Alemán, Griego, Inglés, Español, Persa, Finlandés, Francés, Croata, Húngaro, Indonesio, Italiano, Japonés, Jemer, Coreano, Letón, Bokmal, Holandés, Polaco, Portugués, Ruso, Eslovaco, Esloveno, Albanés, Serbio, Sueco, Turco, Ucraniano y Chino.

El sistema de entrada de teclado requerido se activa automáticamente.

Algunos de estos lenguajes podrían no estar completamente probados, ya que no hablamos todos los lenguajes del mundo. Si encuentras problemas usando uno de los lenguajes supuestamente apoyados, no dudes en decírnoslo. Por ejemplo, probablemente Tails carezca de algunas fuentes o sistemas de entrada no-latinos.

Si quieres que usar Tails sea más fácil para las personas que hablan tu idioma, mira las guías para traductores.
