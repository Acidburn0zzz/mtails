El propósito de este documento es explicar como usar Tails y presentar al usuario sus principales características de seguridad. Está estructurado para ayudarte a completar tareas específicas, además suministra información para ayudarte a entender el tipo de seguridad que Tails provee y sus limitaciones.

Como Tails incluye muchas otras aplicaciones, esta documentación se concentra en los aspectos específicos a Tails. Para aprender más sobre como usar cada aplicación específica, intentamos proveer referencias a documentación externa.

Pero esta documentación no tiene la intención de ser un guía completa de seguridad de computadora. Porque no explica cómo armar una estrategia completa de seguridad, eso dependerá de su caso de uso y modelos de amenaza específicos.

Si estás enfrentando problemas concretos con Tails que no están documentados aquí, vea nuestra sección de Soporte para obtener más ayuda.

Si quieres aprender sobre el funcionamiento interno de Tails, y entender nuestras decisiones de diseño, lee nuestra documentación de proyecto.
