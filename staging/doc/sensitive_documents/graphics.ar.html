Tails includes several applications for image manipulation, graphic design, and page layout:

  • GIMP is an advanced picture editor. You can use it to edit, enhance, and retouch photos and scans, create drawings, and make your own images.

    The GIMP website provides a user manual translated into several languages.

  • Inkscape is vector-based drawing application. You can use it to create a wide variety of graphics such as illustrations, icons, logos, diagrams, maps, and web graphics.

    The Inkscape website provides tutorials translated into several languages.

  • Scribus is a page layout application. You can use it to design newspapers, magazines, newsletters and posters to technical documentation.

    The Scribus website provides documentation in several languages.

These applications can be started from the Applications ▸ Graphics menu.

To learn how to save your working documents encrypted in your Tails USB stick, read our documentation on persistence.

Most files contain metadata which is information characterising the content of the file. For example:

  • Cameras record data about when and where a picture was taken and what camera was used.
  • Office documents automatically add author and company information to texts and spreadsheets.

You can use the MAT to clean the metadata from your files before publishing them. To redact the content of a PDF and strip metadatas from it, you can use pdf-redact-tools .
