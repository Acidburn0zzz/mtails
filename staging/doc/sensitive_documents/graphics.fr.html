Tails inclut de nombreuses applications pour la manipulation d'image, le graphisme, et la mise en page :

  • GIMP est un éditeur avancé d'image. Vous pouvez l'utiliser pour modifier, améliorer et retoucher des photos ou des images scannées, créer des dessins, et faire vos propres images.

    Le site web de GIMP dispose d'un manuel utilisateur traduit dans de nombreuses langues.

  • Inkscape est un éditeur de graphismes vectoriels. Vous pouvez l'utiliser pour créer une grande variété de graphismes comme des illustrations, des icônes, des logos, des diagrammes, des cartes et du graphisme web.

    Le site web d'Inkscape contient des didacticiels traduits dans de nombreuses langues.

  • Scribus est une application de mise en page. Vous pouvez l'utiliser pour faire des journaux, des magazines, des lettres informations et des posters de documentation technique.

    Le site web de Scribus contient de la documentation dans de nombreuses langues.

Ces applications peuvent être lancées depuis le menu Applications ▸ Graphisme.

Pour apprendre comment sauvegarder vos documents de travail de manière chiffrée sur votre clé USB Tails, consultez la documentation relative à la persistance.

La plupart des fichiers contiennent des métadonnées, qui sont des informations à propos du contenu du fichier. Par exemple :

  • Les appareils photo enregistrent des données indiquant où et quand l'image a été prise, et quel appareil photo a été utilisé.
  • Les traitements de texte ajoutent automatiquement aux fichiers le nom de l'auteur et des indications sur son entreprise.

Vous pouvez utiliser MAT pour nettoyer les métadonnées de vos fichiers avant de les publier. Pour éditer le contenu d'un PDF et en retirer les métadonnées, vous pouvez utiliser pdf-redact-tools .
