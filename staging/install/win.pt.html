Instale a partir de outro Tails

If you know someone you trust who uses Tails already, then you can install your Tails by cloning from their Tails.

Você precisa de:

  • 1 outro Tails (USB ou DVD)
  • 1 memória USB (mínimo de 8 GB)
  • ½ hora pra instalar
Instale a partir de outro Tails

Baixe e instale

Você precisa de:

  • 2 USB sticks (at least 8 GB) Why 2?
    X

    Como é atualmente impossível instalar o Tails diretamente a partir do Windows, Linux, macOS, este cenário requer criar um Tails intermediário em uma segunda unidade USB. Este Tails intermediário carece de recursos de segurança e usabilidade importantes.

  • 1 hora para baixar o Tails ( 1.2 GB )
  • 1 hour to install
Install from Windows
You can also:
  • Burn Tails on a DVD
  • Run Tails in a virtual machine
