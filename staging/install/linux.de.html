Von einem anderen Tails aus installieren

Wenn Sie eine Person kennen, der Sie vertrauen und die bereits Tails benutzt, können Sie Ihr Tails durch Kopieren davon installieren.

Sie brauchen:

  • 1 anderes Tails (USB-Stick oder DVD)
  • 1 USB-Stick (mindestens 8 GB)
  • ½ Stunde zum Installieren
Von einem anderen Tails aus installieren

Herunterladen und installieren

Sie brauchen:

  • 2 USB-Sticks (mindestens 8 GB) Warum zwei?
    X

    Da es derzeit nicht möglich ist, Tails direkt von Windows, Linux macOS, aus zu installieren, benötigt diese Vorgehensweise ein zwischenzeitliches Tails auf einem zweiten USB-Stick. Diesem zwischenzeitlichen Tails fehlen wichtige Sicherheits- und Bedienungsfunktionen.

  • 1 Stunde zum Herunterladen von Tails ( 1.2 GB )
  • 1 Stunde zum Installieren
Installation unter Linux
Sie können auch:
  • Tails auf eine DVD brennen
  • Tails in einer virtuellen Maschine laufen lassen
