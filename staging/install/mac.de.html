Spenden

Tails läuft nicht auf jedem Mac-Modell und Sie müssen möglicherweise andere Installationsvorgehensweisen versuchen.

Möglicherweise wird zum Schluss:

  • Tails perfekt funktionieren!
  • es Ihnen möglich sein, Tails mit einigen fehlenden Grundfunktionen zu starten.
  • es Ihnen möglich sein, Tails mit W-LAN- oder Grafikproblemen zu starten.
  • es unmöglich sein, Tails auf Ihrem Mac zu starten.

Nichtsdestotrotz, bereit, einen Versuch zu wagen?

  1. Wenn Sie eine Person kennen, die bereits Tails benutzt und der Sie vertrauen, können Sie versuchen, Tails durch das Kopieren davon zu zu installieren.

    Versuchen Sie von einem anderen Tails zu klonen

  2. Wenn Sie keine Person kennen, die Tails benutzt oder wenn Sie das Klonen versucht haben und das Starten fehlgeschlagen ist, können Sie die folgenden drei Methoden versuchen:

    Installation unter
    Windows oder Linux
    Eine DVD brennen
    und anschließend installieren
    Installation mit
    der Kommandozeile
