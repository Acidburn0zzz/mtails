Ein zwischenzeitliches Tails installieren

In this step, you will install an intermediary Tails by copying the Tails ISO image on the first USB stick using a program called Universal USB Installer. the command line. a program called GNOME Disks.

Bei diesem zwischenzeitlichen Tails können Sie nicht von wichtigen Funktionen, wie automatischen Sicherheitsaktualisierungen oder der Möglichkeit, einige Ihrer Dokumente und Konfigurationen in einem beständigen Speicherbereich zu sichern, profitieren.

Alle Daten auf diesem USB-Stick werden verloren gehen.
