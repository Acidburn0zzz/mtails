Installla una Tails intermediaria

In this step, you will install an intermediary Tails by copying the Tails ISO image on the first USB stick using a program called Universal USB Installer. the command line. a program called GNOME Disks.

Su questa Tails intermediaria non puoi beneficiare di funzionalità importanti, come gli aggiornamenti automatici di sicurezza o la possibilità di salvare documenti e configurazioni sulla partizione cifrata.

Tutti i dati in questa pennetta USB saranno persi.
