Install from another Tails

If you know someone you trust who uses Tails already, then you can install your Tails by cloning from their Tails.

You need:

  • 1 other Tails (USB stick or DVD)
  • 1 USB stick (at least 8 GB)
  • ½ hour to install
Install from another Tails

Download and install

You need:

  • 2 USB sticks (at least 8 GB) Why 2?
    X

    As it is currently impossible to install Tails directly from Windows, Linux, macOS, this scenario requires creating an intermediary Tails on a second USB stick. This intermediary Tails lacks important security and usability features.

  • 1 hour to download Tails ( 1.2 GB )
  • 1 hour to install
