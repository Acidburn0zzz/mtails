اهدا کردن

Tails is a live operating system that you can start on almost any computer from a USB stick or a DVD.

هدف آن محافظت از حریم خصوصی و ناشناسی شماست و به شما کمک می‌کند تا:

  • ناشناس از اینترنت استفاده کرده و سانسور را دور بزنید؛
    تمام اتصالات به اینترنت از طریق شبکهٔ تور انجام شوند؛
  • روی رایانه‌ای که استفاده می‌کنید هیچ ردپایی باقی نمی‌گذارد مگر خودتان صراحتاً درخواست کرده باشید؛
  • از ابزارهای رمزگذاری قوی برای رمزگذاری فایل‌ها، رایانه‌ها و پیغام‌های فوری‌تان استفاده کنید.

در مورد تیلز اطلاعات بیشتری کسب کنید.
