Several security holes that affect Tails 3.6.1 are now fixed in Tails 3.6.2.

We strongly encourage you to upgrade to Tails 3.6.2 as soon as possible.

  • Tor Browser: MSFA 2018-10
  • Thunderbird: Debian Security Advisory 4155
  • libicu: Debian Security Advisory 4150
  • intel-microcode: CVE-2017-5715
