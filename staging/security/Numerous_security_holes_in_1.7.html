Several security holes that affect Tails 1.7 are now fixed in Tails 1.8.1.

We strongly encourage you to upgrade to Tails 1.8.1 as soon as possible.

Details

  • Tor Browser: Mozilla Foundation Security Advisory 2015-134, Mozilla Foundation Security Advisory 2015-138, Mozilla Foundation Security Advisory 2015-139, Mozilla Foundation Security Advisory 2015-145, Mozilla Foundation Security Advisory 2015-146, Mozilla Foundation Security Advisory 2015-147, Mozilla Foundation Security Advisory 2015-149
  • libreoffice: Debian Security Advisory 3394
  • krb5: Debian Security Advisory 3395
  • linux: Debian Security Advisory 3396
  • wpa: Debian Security Advisory 3397
  • libpng: Debian Security Advisory 3399
  • openjdk-7: Debian Security Advisory 3401
  • nspr: Debian Security Advisory 3406
  • dpkg: Debian Security Advisory 3407
  • gnutls26: Debian Security Advisory 3408
  • icedove: Debian Security Advisory 3410
  • openssl: Debian Security Advisory 3413
