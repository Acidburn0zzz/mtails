Several security holes that affect Tails 3.8 are now fixed in Tails 3.9.

We strongly encourage you to upgrade to Tails 3.9 as soon as possible.

  • Tor Browser: Mozilla Foundation Security Advisory 2018-21
  • Thunderbird: Mozilla Foundation Security Advisory 2018-19
  • Linux and intel-microcode:
    • CVE-2018-3620 aka. Foreshadow aka. L1 Terminal Fault
    • too many other CVEs to list them all here
  • CUPS: Debian Security Advisory 4243
  • Exiv2: Debian Security Advisory 4238
  • FUSE: Debian Security Advisory 4257
  • GDM: Debian Security Advisory 4270
  • libsoup: Debian Security Advisory 4241
  • Imagemagick: Debian Security Advisory 4245
  • ffmpeg: Debian Security Advisory 4258, DSA-4249
  • libmspack: Debian Security Advisory 4260
  • Samba: Debian Security Advisory 4271
  • Apache XML Security for C++ library: Debian Security Advisory 4265
  • OpenSSH: Debian Security Advisory 4280
