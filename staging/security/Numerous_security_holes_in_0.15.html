Several security holes affect Tails 0.15.

We strongly urge you to upgrade to Tails 0.16 as soon as possible in case you are still using an older version.

Details

  • iceweasel (MFSA 2012-91, MFSA 2013-01, MFSA 2013-02, MFSA 2013-04, MFSA 2013-05, MFSA 2013-09, MFSA 2013-11, MFSA 2013-12, MFSA 2013-15, MFSA 2013-16, MFSA 2013-17, MFSA 2013-20)
  • perl (Debian Security Advisory 2586)
  • tiff (Debian Security Advisory 2589)
  • ghostscript (Debian Security Advisory 2595)
  • nss (Debian Security Advisory 2599)
  • cups (Debian Security Advisory 2600)
  • gnupg (Debian Security Advisory 2601)
  • libxml2 (Debian Security Advisory 2580)
