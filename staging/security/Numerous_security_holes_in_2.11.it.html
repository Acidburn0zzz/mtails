Several security holes that affect Tails 2.11 are now fixed in Tails 2.12.

We strongly encourage you to upgrade to Tails 2.12 as soon as possible.

  • Tor Browser: Mozilla Foundation Security Advisory 2017-11
  • Linux: 2017-2636
  • jasper: Debian Security Advisory 3827
  • gstreamer: Debian Security Advisory 3818, Debian Security Advisory 3819, Debian Security Advisory 3820, Debian Security Advisory 3821, Debian Security Advisory 3822
  • eject: Debian Security Advisory 3823 - imagemagick: Debian Security Advisory 3808 - pidgin: Debian Security Advisory 3806 - samba: Debian Security Advisory 3816 - Fix hardlink permissions on /var/tmp: #12125 - Ensure /etc/resolv.conf is owned by root:root.
