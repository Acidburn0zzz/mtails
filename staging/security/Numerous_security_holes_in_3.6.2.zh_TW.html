Several security holes that affect Tails 3.6.2 are now fixed in Tails 3.7.

We strongly encourage you to upgrade to Tails 3.7 as soon as possible.

  • Tor Browser: MSFA 2018-12
  • OpenSSL: Debian Security Advisory 4157
  • Perl: Debian Security Advisory 4172
  • Libre Office: Debian Security Advisory 4178
  • libmad: Debian Security Advisory 4192
