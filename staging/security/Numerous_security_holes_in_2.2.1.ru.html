Several security holes that affect Tails 2.2.1 are now fixed in Tails 2.3.

We strongly encourage you to upgrade to Tails 2.3 as soon as possible.

Details

  • Tor Browser: Mozilla Foundation Security Advisory 2016-39, Mozilla Foundation Security Advisory 2016-44, Mozilla Foundation Security Advisory 2016-47
  • icedove: Debian Security Advisory 3520
  • git: Debian Security Advisory 3521
  • libgd3: Debian Security Advisory 3556
  • pidgin-otr: Debian Security Advisory 3528
  • srtp: Debian Security Advisory 3539
  • imagemagick: Debian Security Advisory 3547
  • samba: Debian Security Advisory 3548
  • openssh: Debian Security Advisory 3550
