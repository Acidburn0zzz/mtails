Several security holes that affect Tails 1.6 are now fixed in Tails 1.7.

We strongly encourage you to upgrade to Tails 1.7 as soon as possible.

Details

  • Tor Browser: Mozilla Foundation Security Advisory 2015-116, Mozilla Foundation Security Advisory 2015-122, Mozilla Foundation Security Advisory 2015-123, Mozilla Foundation Security Advisory 2015-127, Mozilla Foundation Security Advisory 2015-128, Mozilla Foundation Security Advisory 2015-130, Mozilla Foundation Security Advisory 2015-131, Mozilla Foundation Security Advisory 2015-132, Mozilla Foundation Security Advisory 2015-133
  • wget: CVE-2015-7665
  • libfreetype6: Debian Security Advisory 3370
  • gdk-pixbuf: Debian Security Advisory 3378
  • Linux: Debian Security Advisory 3372
  • openjdk-7: Debian Security Advisory 3381
  • unzip: Debian Security Advisory 3386
  • Prevent a symlink attack on ~/.xsession-errors via tails-debugging-info which could be used by the amnesia user to read the contents of any file, no matter the permissions.
