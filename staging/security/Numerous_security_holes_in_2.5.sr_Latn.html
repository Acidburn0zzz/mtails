Several security holes that affect Tails 2.5 are now fixed in Tails 2.6.

We strongly encourage you to upgrade to Tails 2.6 as soon as possible.

Details

  • Tor Browser: Mozilla Foundation Security Advisory 2016-86
  • icedove: Debian Security Advisory 3647
  • linux: Debian Security Advisory 3659
  • curl: Debian Security Advisory 3638
  • openjdk-7: Debian Security Advisory 3641
  • fontconfig: Debian Security Advisory 3644
  • gnupg: Debian Security Advisory 3649
  • libgcrypt20: Debian Security Advisory 3650
  • imagemagick: Debian Security Advisory 3652
  • libarchive: Debian Security Advisory 3657
  • libidn: Debian Security Advisory 3658
