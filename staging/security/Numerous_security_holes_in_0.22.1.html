Several security holes affect Tails 0.22.1.

We strongly encourage you to upgrade to Tails 0.23 as soon as possible.

Details

  • Iceweasel and its bundled NSS: Mozilla Foundation Security Advisory 2014-15, Mozilla Foundation Security Advisory 2014-16, Mozilla Foundation Security Advisory 2014-17, Mozilla Foundation Security Advisory 2014-19, Mozilla Foundation Security Advisory 2014-26, Mozilla Foundation Security Advisory 2014-27, Mozilla Foundation Security Advisory 2014-28, Mozilla Foundation Security Advisory 2014-29, Mozilla Foundation Security Advisory 2014-30, Mozilla Foundation Security Advisory 2014-31
  • Linux kernel: 2014-1438, 2013-4579
  • Iceweasel
  • cups: Debian Security Advisory 2876
  • mutt: Debian Security Advisory 2874
  • file: Debian Security Advisory 2861, Debian Security Advisory 2873
  • udisks: Debian Security Advisory 2872
  • libyaml-libyaml-perl: Debian Security Advisory
  • gnutls26: Debian Security Advisory 2869
  • libtar: Debian Security Advisory 2863
