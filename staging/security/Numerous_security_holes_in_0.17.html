Several security holes affect Tails 0.17.

We strongly urge you to upgrade to Tails 0.17.1 as soon as possible in case you are still using an older version.

Details

  • Iceweasel (MFSA 2013-29)
  • Linux (CVE-2013-0871, CVE-2013-0217)
  • Perl (Debian Security Advisory 2641)
  • openjpeg (Debian Security Advisory 2629)
  • sudo (Debian Security Advisory 2642)
