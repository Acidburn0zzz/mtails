Several security holes that affect Tails 1.4 are now fixed in Tails 1.4.1.

We strongly encourage you to upgrade to Tails 1.4.1 as soon as possible.

Details

  • Tor Browser: Mozilla Foundation Security Advisory 2015-71, Mozilla Foundation Security Advisory 2015-70, Mozilla Foundation Security Advisory 2015-69, Mozilla Foundation Security Advisory 2015-66, Mozilla Foundation Security Advisory 2015-65, Mozilla Foundation Security Advisory 2015-64, Mozilla Foundation Security Advisory 2015-61, Mozilla Foundation Security Advisory 2015-59
  • Linux: Debian Security Advisory 3290, CVE-2015-3636
  • p7zip: Debian Security Advisory 3289
  • OpenSSL: Debian Security Advisory 3287, CVE-2014-8176, CVE-2015-1788, CVE-2015-1789, CVE-2015-1790, CVE-2015-1791, CVE-2015-1792, CVE-2015-4000
  • CUPS: Debian Security Advisory 3283, CVE-2015-1158, CVE-2015-1159
  • ntfs-3g: Debian Security Advisory 3268, CVE-2015-3202
  • FUSE: Debian Security Advisory 3266, CVE-2015-3202
