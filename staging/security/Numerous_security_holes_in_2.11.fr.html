Plusieurs failles de sécurité qui affectent Tails 2.11 sont maintenant corrigées dans Tails 2.12.

Nous vous encourageons vivement à mettre à jour vers Tails 2.12 dès que possible.

  • Navigateur Tor : Mozilla Foundation Security Advisory 2017-11
  • Linux : 2017-2636
  • jasper : Debian Security Advisory 3827
  • gstreamer : Debian Security Advisory 3818, Debian Security Advisory 3819, Debian Security Advisory 3820, Debian Security Advisory 3821, Debian Security Advisory 3822
  • eject : Debian Security Advisory 3823
  • imagemagick : Debian Security Advisory 3808
  • pidgin : Debian Security Advisory 3806
  • samba : Debian Security Advisory 3816
  • Correction des permissions de lien matériel sur /var/tmp : #12125
  • S'assurer que /etc/resolv.conf appartient à root:root.
