Several security holes that affect Tails 1.5 are now fixed in Tails 1.5.1.

We strongly encourage you to upgrade to Tails 1.5.1 as soon as possible.

Details

  • Tor Browser: Mozilla Foundation Security Advisory 2015-94, Mozilla Foundation Security Advisory 2015-95
  • NSS: Debian Security Advisory 3336
  • gdk-pixbuf: Debian Security Advisory 3337
