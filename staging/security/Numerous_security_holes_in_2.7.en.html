Several security holes that affect Tails 2.7 are now fixed in Tails 2.7.1.

We strongly encourage you to upgrade to Tails 2.7.1 as soon as possible.

Details

  • Tor Browser: Mozilla Foundation Security Advisory 2016-92
  • gstreamer0.10: Debian Security Advisory 3717, Debian Security Advisory 3726
  • icu: Debian Security Advisory 3725
  • imagemagick: Debian Security Advisory 3726
  • vim: Debian Security Advisory 3722
