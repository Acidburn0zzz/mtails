Several security holes that affect Tails 3.7.1 are now fixed in Tails 3.8.

We strongly encourage you to upgrade to Tails 3.8 as soon as possible.

  • Tor Browser: Mozilla Foundation Security Advisory 2018-17
  • Enigmail: CVE-2018-12020, CVE-2018-12019
  • libgcrypt Debian Security Advisory 4231
  • perl Debian Security Advisory 4226
