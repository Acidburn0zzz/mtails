Plusieurs failles de sécurité qui affectent Tails 3.0 sont maintenant corrigées dans Tails 3.0.1.

Nous vous encourageons vivement à mettre à jour vers Tails 3.0.1 dès que possible.

  • tor : TROVE-2017-006 (alias bug #22753 on Tor Project's Trac et 2017-0377)
  • Linux : Debian Security Advisory 3886, incluant 2017-1000364 alias Stack Clash
  • libc : Debian Security Advisory 3887 (alias Stack Clash)
  • libexpat1 : Debian Security Advisory 3898
  • libgcrypt20 : Debian Security Advisory 3901
  • libgnutls30 : Debian Security Advisory 3884
