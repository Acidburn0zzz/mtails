Several security holes that affect Tails 2.9.1 are now fixed in Tails 2.10.

We strongly encourage you to upgrade to Tails 2.10 as soon as possible.

Details

  • Tor Browser: Mozilla Foundation Security Advisory 2017-02
  • bind9: Debian Security Advisory 3758
  • icedove: Debian Security Advisory 3757
  • pcsc-lite: Debian Security Advisory 3752
  • libgd2: Debian Security Advisory 3751
  • libxml2: Debian Security Advisory 3744
  • tor: Debian Security Advisory 3741
  • samba: Debian Security Advisory 3740
