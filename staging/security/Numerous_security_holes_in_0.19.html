Several security holes affect Tails 0.19.

We strongly urge you to upgrade to Tails 0.20 as soon as possible in case you are still using an older version.

Details

  • Iceweasel ( MFSA 2013-63, MFSA 2013-68, MFSA 2013-69, MFSA 2013-72, MFSA 2013-73, MFSA 2013-75 )
  • libgcrypt11 (Debian Security Advisory 2731)
  • gnupg (Debian Security Advisory 2730)
  • bind9 (Debian Security Advisory 2728)
  • openjdk-6 (Debian Security Advisory 2727)
  • poppler (Debian Security Advisory 2719)
  • curl (Debian Security Advisory 2713)
