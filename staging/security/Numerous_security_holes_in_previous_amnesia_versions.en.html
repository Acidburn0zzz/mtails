The following security holes affect amnesia systems older than 0.5.

  • polipo: DSA-2002
  • Linux kernel: DSA-2003, DSA-2012, DSA-1996
  • cups: DSA-2007
  • dpkg: DSA-2011
  • curl: DSA-2023
  • iceweasel: DSA-2027, DSA-1999
  • libpng: DSA-2032
  • pidgin: DSA-2038
  • openoffice.org: DSA-1995
